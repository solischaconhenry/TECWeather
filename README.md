# TECWeather
Estación Meteorológica del TEC, Santa Clara, Alajuela, Costa Rica

Estación capáz de tomar los datos:

1. Cantidad de Lluvia
2. Presion Atmosférica
3. Temperatura C°, F°
4. Luminosidad
5. Humedad
6. Dirección del Viento
7. Velocidad del Viento
